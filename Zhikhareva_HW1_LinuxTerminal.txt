Linux terminal (GitBash) commands

1) Посмотреть где я: 

pwd

2) Создать папку: 

mkdir papka

3) Зайти в папку: 

cd papka

4) Создать 3 папки: 

mkdir p1 p2 p3

5) Зайти в любую папку: 

cd p2

6) Создать 5 файлов (3 txt, 2json): 

touch 1t.txt 2t.txt 3t.txt 1js.json 2js.json

7) Создать 3 папки:

mkdir f1 f2 f3

8) Вывести список содержимого папки:

ls -la 

9) +открыть любой txt файл:

vim 3t.txt 

10)+написать туда что-нибудь, любой текст

i
текст

11)+сохранить и выйти:

esc
:wq

12) Выйти из папки на уровень выше

cd ..

13) Переместить любые 2 файла, которые вы создали  в любую другую папку

mv ./p2/{1t.txt. 2j.json} ./p3/

14) Скопировать любые 2 файла, которые вы создали в любую другую папку

cp ./p3/{1t.txt. 2j.json} ./p1/

15) Найти файл по имени

find -name 2t.txt

16) Посмотреть содержимое в реальном времени (grep) и изучить как она работает

tail -f 3t.txt | grep текст

17) Вывести несколько первых строк из текстового файла

head -n 3t.txt #n - количесвто строк

18) Вывести несколько последних строк из текстового файла

tail -n 3t.txt #n - количесвто строк

19) просмотреть содержимое длинного файла (команда Less) изучить как она работает

less 3t.txt # f — вперёд на страницу, b — назад на страницу, q -выход, 
                    /текст - поиск в тексте (n- переход к следующему совпадению,N - переход к предыдущему совпадению).

20) вывести дату и время

date
